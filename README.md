# IT-Tuki-ja-Kyberturva-Tyokalut
Projektit:  Automaattinen Varmuuskopiointijärjestelmä (Järjestelmätuki): Python-skripti, joka luo aikaleimattuja varmuuskopioita tärkeistä tiedostoista.  Vahva Salasana Generaattori (Kyberturvallisuus): Työkalu, joka luo turvallisia salasanoja käyttämällä satunnaisia merkkejä, numeroita ja erikoismerkkejä.
